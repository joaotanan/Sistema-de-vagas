# Sistema de Vagas 💼

Projeto full stack simples para cadastro de vagas.

## Tecnologias
- Node.js
- Express
- HTML, CSS, JavaScript

## Como rodar

### Backend
cd backend
npm init -y
npm install express
node server.js

### Frontend
Abrir index.html com Live Server

## Funcionalidades
- Criar vagas
- Listar vagas
- Deletar vagas
